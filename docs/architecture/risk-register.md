# Risk Register

## 기술 리스크

### R-001: better-sqlite3 네이티브 모듈 배포 이슈
- **심각도**: 높음
- **가능성**: 높음
- **현재 상태**: 발생 (빌드 시 generateStaticParams 충돌)
- **대응**: force-dynamic 설정으로 SSR 전환, Turso(LibSQL) 마이그레이션 예정
- **장기 해결**: Drizzle + Turso HTTP driver로 전환 시 Edge Runtime 호환

### R-002: Node.js 25 호환성
- **심각도**: 중간
- **가능성**: 낮음
- **대응**: 주요 패키지 호환성 검증 완료, package.json engines 설정

## 일정 리스크

### R-003: 기능 범위 확장

- **가능성**: 중간
